# 中国车牌模拟生成器

一个基于 Python 和 OpenCV 的中国机动车号牌生成工具，支持多种类型的车牌号码和样式。本项目严格按照 GA 36-2018 标准实现，适用于数据增强、机器学习训练数据生成等场景。

## ✨ 特性

- 🚗 **多种车牌类型支持**：普通车牌、新能源车牌、特种车牌
- 🎨 **多样化颜色样式**：蓝牌、绿牌、黄牌、白牌、黑牌
- 📏 **单双层车牌**：支持单层（140mm）和双层（220mm）车牌
- 🎯 **高度真实**：基于标准字体库和真实车牌底板尺寸
- 🔀 **随机生成**：支持大批量随机车牌生成
- 🎮 **定制生成**：支持指定号码、颜色、样式的车牌生成

## 🚀 支持的车牌类型

### 按颜色分类
- **蓝色车牌**：普通小型汽车
- **黄色车牌**：大型汽车、挂车、教练车
- **绿色车牌**：新能源汽车（轿车/卡车）
- **白色车牌**：警用车辆、军用车辆
- **黑色车牌**：使领馆车牌、港澳入境车辆

### 按尺寸分类
- **单层车牌**：440×140mm 或 480×140mm（新能源）
- **双层车牌**：440×220mm（挂车等）

## 🎯 使用方法

### 1. 随机批量生成车牌

```bash
python generate_multi_plate.py --number 100 --save-adr output_folder
```

**参数说明：**
- `--number`: 生成车牌数量（默认: 10）
- `--save-adr`: 保存路径（默认: multi_val）

### 2. 生成指定车牌

```bash
python generate_special_plate.py --plate-number 湘A12345 --double False --bg-color blue
```

**参数说明：**

| 参数 | 说明 | 示例 |
|------|------|------|
| `--plate-number` | 车牌号码 | 湘A12345, 京AD88888 |
| `--double` | 是否双层车牌 | True/False |
| `--bg-color` | 底板颜色 | blue, green_car, yellow 等 |

## 🔧 核心功能

### MultiPlateGenerator 类

主要生成器类，提供以下功能：

- `generate_plate()`: 随机生成车牌
- `generate_plate_special()`: 生成指定样式车牌
- `get_location_multi()`: 获取字符位置信息

### 车牌号码生成规则

根据不同车牌类型实现标准化生成：

- **普通车牌**: 7位字符（省份+字母+5位数字/字母）
- **新能源车牌**: 8位字符（特殊字母标识）
- **特种车牌**: 包含特殊汉字（警、使、领、港、澳等）

## ⚠️ 重要说明

1. **合规使用**: 请确保生成的车牌号码符合交通法规，避免使用真实存在的车牌号码
2. **用途限制**: 本工具仅供学习、研究和数据生成使用，不得用于违法用途
3. **格式要求**: 不同类型车牌有特定的位数要求：
   - 普通车牌：7位
   - 新能源车牌：8位
   - 军用车牌：字母开头的7位字符
